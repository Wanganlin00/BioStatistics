11  主成分分析

  • Show All Code
  • Hide All Code
  • View Source

主成分分析(Principal component analysis，PCA) 是一种线性降维方法。

对于一组变量\(X_1,X_2,...,X_p\),存在它们的线性组合\(PC\)，令\(var(PC)\)最大，得到\(PC_1\),再找\(PC_2\)，\(PC_2\)与\(PC_1\)正交，以此类推，找到一组\(PC_1,PC_2,...,PC_m\)无关主成分。

# 主成分分析

主成分分析(Principal component analysis，PCA) 是一种线性降维方法。

对于一组变量$X_1,X_2,...,X_p$,存在它们的线性组合$PC$，令$var(PC)$最大，得到$PC_1$,再找$PC_2$，$PC_2$与$PC_1$正交，以此类推，找到一组$PC_1,PC_2,...,PC_m$无关主成分。
